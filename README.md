La recherche séquentielle

Principe : 

La recherche séquentielle d’un élément dans une séquence, consiste à vérifier l’existence de l’élément dans la séquence, par le fait de parcourir la séquence élément par élément jusqu’à trouver ce dernier ou arriver à parcourir tous les éléments de la séquence.
